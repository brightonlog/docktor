# 📌 프로젝트 기여 및 학습 내용 정리

## 1. 데이터 기반 아이디어 기획 도출

본 프로젝트는 **실질적인 데이터 활용 가능성**을 중심으로 아이디어를 기획하는 과정에서 시작되었다.  
AI Hub에서 제공하는 선박 데이터를 조사하던 중, **선박 외판 도장 상태를 이미지 기반으로 평가할 수 있는 데이터셋**이 존재함을 확인하였고, 이를 계기로 프로젝트의 방향성을 **선박 도장 품질 검사 자동화**로 설정하게 되었다.

단순한 아이디어 제안에 그치지 않고,  
이미 공개 데이터가 존재하며 AI 적용이 가능한 영역을 기반으로 기획함으로써 프로젝트의 **현실성·확장성·구현 가능성**을 함께 고려한 기획 역량을 기를 수 있었다.

---

## 2. 현장 조사 및 실제 검사 방식 분석

아이디어의 타당성을 검증하기 위해 조선업에 재직 중인 실무자를 대상으로 인터뷰를 진행하였으며, 이를 통해 현재 외판 선체 검사가 다음과 같은 방식으로 이루어지고 있음을 파악하였다.

- 크레인 바스켓에 선주, 감독관, 검사원, 조선소 검사부 직원 등 다수 인원이 동승
- 크레인을 리모컨으로 조작하여 외판 특정 구역에 접근
- 사람이 직접 육안으로 결함을 확인
- 결함 여부를 구두 또는 기록으로 공유
- 모든 검사 공정은 자동화 없이 전적으로 사람의 판단에 의존

이를 통해 외판 선체 검사는  
**고소 작업으로 인한 안전 위험**, **검사 결과의 주관성**, **비효율적인 인력 구조**라는 명확한 한계를 지니고 있음을 인식하게 되었다.

---

## 3. 실제 선박 방문을 통한 데이터 수집 경험

AI Hub 데이터에 대한 이해를 심화하기 위해 실제 선박을 직접 확인할 수 있는 장소를 방문하여,  
AI Hub에서 확인한 도장 품질 데이터가 **현장에서는 어떤 형태로 나타나는지**를 관찰하고 촬영하는 과정을 수행하였다.

이 과정에서 다음과 같은 경험을 얻었다.

- 도장 손상, 표면 불균일, 오염 등이 실제 선박 외판에서 어떻게 보이는지 직접 확인
- 공개 데이터와 실제 현장 이미지 간의 차이점(조명, 거리, 각도, 표면 반사 등)을 체감
- 향후 AI 모델 학습 및 테스트에 활용 가능한 **실제 환경 기반 이미지 데이터** 직접 수집

이를 통해 **데이터를 제공받아 사용하는 입장이 아닌, 직접 확보하고 해석하는 경험**을 쌓을 수 있었으며,  
AI 모델 설계 시 실제 환경을 고려하는 것이 필수적이라는 인식을 갖게 된 중요한 계기가 되었다.

---

## 4. 아이디어 구체화 및 시스템 방향 설정

현장 조사와 데이터 분석을 바탕으로, 검사원이 선박 외판 상태를 보다 직관적으로 파악할 수 있도록  
**웹 기반 3D 시각화 검사 지원 시스템**으로 아이디어를 구체화하였다.

실제 선박 구조를 반영한 3D 모델 위에서  
- 결함 위치를 한눈에 확인하고  
- 검사 결과를 시각적으로 이해할 수 있는 구조  

를 목표로 시스템 방향을 설정하였다.

이를 위해 **Three.js 기반 3D 시각화 기술**을 도입하여, 단순한 시각적 표현이 아닌  
**검사 업무를 보조하는 인터페이스로서의 3D 활용 가능성**을 탐색하였다.

---

## 5. 기술 스택 학습 및 구현 경험

Three.js를 활용한 선박 3D 시각화를 구현하며 웹 기반 3D 그래픽스 전반에 대해 폭넓게 학습하였다.  
이론에 그치지 않고 실제 구현 과정에서 다양한 문제를 직접 해결하며 다음과 같은 핵심 역량을 습득하였다.

### 🔹 3D 그래픽스 이론
- 좌표계, 카메라, 렌더링 파이프라인 이해
- PBR(Physically Based Rendering) 조명 모델 학습

### 🔹 React + Three.js 통합
- 선언적 3D 프로그래밍 패턴
- 컴포넌트 기반 씬(Scene) 구성

### 🔹 성능 최적화
- 메모이제이션을 활용한 렌더링 최적화
- 60fps 유지를 위한 구조 개선

### 🔹 문제 해결 역량
- 렌더링 오류 및 성능 이슈에 대한 체계적 디버깅
- 기술적 한계 인식 및 대안 탐색

### 🔹 사용자 중심 개발
- 실제 검사 흐름을 고려한 UX 설계
- 직관적인 인터랙션을 위한 반복적 개선

## 6. 프로젝트를 통해 얻은 성과와 개인적 기여

본 프로젝트에서 다음과 같은 역할을 수행하며 프로젝트의 방향성과 현실성을 강화하는 데 기여하였다.

- AI Hub 데이터 기반 프로젝트 기획
- 조선업 실무자 인터뷰를 통한 문제 정의
- 실제 선박 방문 및 현장 이미지 데이터 수집
- 웹 기반 3D 시각화 기술 도입 및 검증

특히 기술 구현 이전에 **현장을 이해하고 데이터를 해석하는 과정의 중요성**을 체감하며,  
기술 중심 사고에서 벗어나 **현실 문제 해결 중심의 개발 사고로 성장할 수 있는 계기**가 되었다.
